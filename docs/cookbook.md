## üõ†Ô∏è How to Run

To run these notebooks locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Hawksight-AI/semantica.git
    cd semantica
    ```

2.  **Install dependencies**:
    ```bash
    pip install -e .[all]
    pip install jupyter
    ```

3.  **Launch Jupyter**:
    ```bash
    jupyter notebook
    ```
